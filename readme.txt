Terminal Connect 4.
Name: Ty Crabtree

Description:
	This is an implementation of Connect 4 with artificial intelligence. This game is ran and played in the terminal window. 

Building / Running;
	1. Open terminal.
	2. Install Tkinter. For Linux mint (ubuntu systems) use "sudo apt-get install python-tk"
	2. Use "python c4GUI.py" to run the code (ensure that this command is ran in the directory with the c4.py)
	3. Enjoy.
